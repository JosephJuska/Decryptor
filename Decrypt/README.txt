----This text file gives information about the Decryptor.----

Name: Decryptor
Description: Decrypts the text inside the given path
Made by: Yusif

----How to use?----

Values and uses:

-t, --text: Path for the text file which will be decrypted. Will give error if value in text can't be found in the base of decryption. Please look at the section "Bases of decryption.

-p, --path: Path where decrypted file will be stored. This text file or the path should be empty.

-n, --number: Rotation number for decryption. Can be any range. Can be activated with a key.

-k, --key: Key for decryption. Should be given correctly for the base of decryption(uppercase letters or only letters).

Will give error if value in key can't be found in the base of decryption. Please look at the section "Bases of decryption".Can be activated with a rotation number.

-uc, --upper_case: Option for the base of decryption. Can't be used with letters_only. To know about this base please look at the section "Bases of decryption".

-lo, --letters_only: option for the base of decryiption. to know about this base please look at the section "Bases of decryption".
